# 简要使用手册

## 功能简介

将要处理的数据以所给格式填入data.txt文件中后运行可执行程序，终端将输出部分统计结果以供使用，同时结果将记录入result.txt文件中.

## data.txt输入示例

注:**所有的**空格与标点都是严格的格式化输入的一部分，自己写的简陋函数不支持多种格式的同时识别，有需要可以自己更改.c与.h文件中farrays函数中的标志符参数等.

### 输入文本

```test
mode = 0
num_list = 2
num_measure = [4,4]
num_b = [2,5]
x1 = [0.233599,0.283889,0.334319,0.384169];u_b = 0.001
x2 = [2.31101,2.79772,3.28168,3.76410];u_b = 0.00001
alpha = [1,-1]
```

注:这是大物弹簧振子实验的mi与Ti^2的数据，理论表达式为Ti^2=4π^2/k·(mi+m0),对应四种不同滑块重量的情况，因此下文multi模式示例输出无实际意义，主要用来计算线性回归.

### 参数含义

**mode**:处理模式,取值为{single=0,multi=1,link=2};

|mode取值|模式说明|
|----|----|
|single|单变量处理模式，计算xi的各种观测值|
|multi|当结果导出式g(xi)为形如g=Const*x_1^alpha_1*x_2^alpha_2*...*x_n^alpha_n乘积表达式时可使用，直接给出g的相对不确定度|
|link|一元线性回归计算，输出相关系数，回归系数，以及数据分布导致的不确定度|
|differ|逐差法数据处理，如果数据有奇数个会舍去最后一个|

**num_list**：所测量的数据组数量，即多少列xi;
**num_measure**：各组测量数据数量，也就是每个xi有多少测量值，以列表形式给出，数值个数应等于num_list;
**num_b**：单次测量值仪器测量次数，也就是每测量一次xi使用多少次仪器，造成了多少次测量误差，以列表形式给出，数值个数应等于num_list;
**xi**：测量数据列表，以列表形式给出，数值个数应等于num_measure; u_bi：xi测量仪器的B类不确定度;两者在同一行被读取;
**alpha**：模式multi积式指数，以列表形式给出，数值个数应等于num_list;
